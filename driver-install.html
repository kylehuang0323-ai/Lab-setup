⚡ 一键安装（快速开始）

方式一：从 NAS 直接运行

REM 在工作站上以管理员身份打开 CMD，执行： \\192.168.0.81\Software\AXB35_02_Win11_24H2_Driver_list_V007\AllDriverInstall.cmd

方式二：复制到本地后运行（推荐，更快更稳定）

REM 1. 映射 NAS net use Z: \\192.168.0.81\Software REM 2. 复制驱动包到本地 xcopy Z:\AXB35_02_Win11_24H2_Driver_list_V007 C:\Drivers\AXB35 /E /I /H /Y REM 3. 运行一键安装脚本（自动请求管理员权限） C:\Drivers\AXB35\AllDriverInstall.cmd

方式三：使用优化版脚本（带日志和进度显示）

REM 将优化版脚本复制到驱动包根目录后运行： copy \\192.168.0.81\Software\GMK-EVO-X2-DriverInstall.cmd C:\Drivers\AXB35\ C:\Drivers\AXB35\GMK-EVO-X2-DriverInstall.cmd REM 安装完成后查看日志： type C:\Drivers\AXB35\DriverInstall_*.log
⚠️
注意事项：
• 必须以管理员身份运行（脚本会自动提权）
• 安装过程中屏幕可能闪烁（显卡驱动安装时正常）
• 全部完成后务必重启计算机
• 预计安装时间：8-15 分钟

💻 PowerShell 一键脚本（从 NAS 安装）

适合已连接 NAS 的工作站，一行命令完成全部安装：

# 以管理员身份运行 PowerShell，执行： $drvRoot = "\\192.168.0.81\Software\AXB35_02_Win11_24H2_Driver_list_V007" # 复制到本地（网络安装可能被策略阻止） Copy-Item $drvRoot -Destination "C:\Drivers\AXB35" -Recurse -Force Set-Location "C:\Drivers\AXB35" # 按顺序安装所有驱动 $drivers = @( @{Folder="01_AMD_Chipset_7.04.30.030"; Script="Install.bat"; Name="AMD Chipset"}, @{Folder="02_AMD_Graphics_24.20.64"; Script="Install.bat"; Name="AMD Graphics"}, @{Folder="03_Senary_Audio_4.18.0.4"; Script="Install.bat"; Name="Senary Audio"}, @{Folder="04_AMD_WiFi_5.5.0.3760"; Script="wifiInstall.bat"; Name="AMD WiFi"}, @{Folder="05_AMD_BT_1.1042.0.527"; Script="BTInstall.bat"; Name="AMD Bluetooth"}, @{Folder="06_Lan_1125.021.0903.2024"; Script="Install.bat"; Name="Realtek LAN"}, @{Folder="07_Cardreader_1.1.49.0"; Script="Install.bat"; Name="Card Reader"}, @{Folder="08_ACP_7.0.0.44"; Script="install.bat"; Name="AMD ACP"}, @{Folder="09_Camera_11.4.2.629"; Script="install.bat"; Name="Camera"}, @{Folder="10_AMD_radeonsoftware_14.0.33728.0"; Script="Appinstall.bat"; Name="Radeon Software"} ) foreach ($drv in $drivers) { Write-Host "[安装中]" $drv.Name -ForegroundColor Cyan Push-Location $drv.Folder & cmd.exe /c $drv.Script 2>&1 | Out-Null if ($LASTEXITCODE -eq 0) { Write-Host " [成功]" $drv.Name -ForegroundColor Green } else { Write-Host " [失败]" $drv.Name "(Exit: $LASTEXITCODE)" -ForegroundColor Red } Pop-Location Start-Sleep -Seconds 2 } Write-Host "`n全部完成！建议重启计算机。" -ForegroundColor Yellow # Restart-Computer -Force # 取消注释以自动重启

🚨 故障排除

问题可能原因解决方案
安装脚本闪退 未以管理员运行 右键 → 以管理员身份运行
显卡驱动安装失败 需先安装芯片组驱动 确保按 01→10 顺序安装
网卡驱动安装后无网络 需要重启生效 重启计算机
NAS 路径无法访问 网卡驱动未装/IP 未配 先用 U 盘拷贝有线网卡驱动(06)手动安装
设备管理器仍有感叹号 驱动未完整安装 检查日志 → 重新运行对应单个驱动的 bat
💡
最佳实践：如果工作站还未联网（新装系统），建议先用 U 盘拷贝 06_Lan（有线网卡）驱动手动安装，连上 NAS 后再运行完整的一键脚本。
